    Welcome to HyperSpec

    This is a project developed by me, (@crobisaur). Disclaimer: This page is currently under development and may be poorly formatted or incorrect.

    What is it?

    HyperSpec is a collection of scripts written to transform non-standard binary sequential (BSQ) data into simple to use numpy arrays. From there several other packages were implemented to analyze and classify hyperspectral data.
